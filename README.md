# 🔐 Random Password Generator

A simple and interactive password generator built using Python and Streamlit.

This application allows users to generate strong passwords with customizable options such as password length and character types.

## Features

- Generate secure random passwords
- Choose password length
- Include letters, numbers, and symbols
- Exclude specific characters
- Generate multiple passwords at once
- Simple and clean user interface

## Technologies Used

- Python
- Streamlit

## Installation

1. Clone the repository

git clone https://github.com/yourusername/password-generator-streamlit.git

2. Navigate into the project folder

cd password-generator-streamlit

3. Install dependencies

pip install -r requirements.txt

4. Run the application

streamlit run password_app.py

## Example Output

The application generates strong passwords such as:

G7$k9!Pq2@Zx
mQ4&Lz8#Np3!
